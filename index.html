语法知识

static

  • static 修饰局部变量的作用是让局部变量只初始化一次，局部变量在内存中只有一份。

  • static 修饰全局变量，全局变量的作用域仅限于当前文件。

copy

  • copy 返回不可变的字符串对象，mutableCopy返回可变的字符串对象。

  • property里面的copy相当于set方法中使用字符串的copy设置该属性值。

const

  • 将项目中的所有常量定义在一个ICIConst.h 中，在.m文件中定义常量的值.因为OC修改字符串使用ICIName,和c语言不同，不是使用 *ICIName，所以将const来修饰 ICIName.

    const int p1 = NULL;// p不可以修改，p可以修改
    int const p2 = NULL;// 这句和上面的作用一样
    int     const p3 = NULL;// p可以修改，p不可以修改，
    int a1 = 10；     p1 = 20;//错误
    p = &a;//正确
    *p3 = 20;//正确
    p3 = &a;//错误



int numbers[4] = {10,20,30,40};
//number是相当于&numbers[0]，指向数组首元素的指针
int *p = (int *)(&numbers + 1);//&numbers相当于指向数组的一个指针，+1就相当于+指针        指向数组的字节数，为16；

因为view是懒加载的，所以第一次使用到的时候在view的get函数中会首先调用

self.navigationItem.title = @"标题";

self.tableView.tableFooterView = [UIView alloc] init];

    - (UIView *)hitTest:(CGPoint)point withEvent:(UIEvent *)event
    {
    //    NSLog(@"%@--hitTest",[self class]);
    //    return [super hitTest:point withEvent:event];


    // 1.判断当前控件能否接收事件
    if (self.userInteractionEnabled == NO || self.hidden == YES || self.alpha <= 0.01) return nil;

    // 2. 判断点在不在当前控件
    if ([self pointInside:point withEvent:event] == NO) return nil;

    // 3.从后往前遍历自己的子控件
    NSInteger count = self.subviews.count;
    for (NSInteger i = count - 1; i >= 0; i--) {
    UIView *childView = self.subviews[i];
    // 把当前控件上的坐标系转换成子控件上的坐标系
     CGPoint childP = [self convertPoint:point toView:childView];

       UIView *fitView = [childView hitTest:childP withEvent:event];
    if (fitView) { // 寻找到最合适的view
        return fitView;
    }        
}
    // 循环结束,表示没有比自己更合适的view
    return self;
}
