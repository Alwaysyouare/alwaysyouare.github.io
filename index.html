iOS9新特性

nonnull

  • 新的修饰符nonnull nullable null_resettable,其作用：

  • 泛型.类似Java的泛型定义

  • __kindof关键字会告诉编译器返回的可能是某种类型或者其子类型的数据

UIWindow

  • iOS9后任何的UIWindow都需要一个根控制器。并且状态栏是由UIWindow的顶层控制器来决定的。
  • 可以自定义UIWindow的UIViewController来实现状态栏的控制，或者在自定义的touchesBegan中处理UIWindow的点击事件。

RunLoop

  • RunLoop 就是运行循环；

    • 实质上就是在一个do..while循环中处理各种任务；
    • 一个线程对应一个RunLoop，调用 currentRunLoop会获取当前线程的RunLoop；
    • 主线程的RunLoop会自动启动，其他线程的需要自己启动run；
    • RunLoop只能使用一个模式启动。

  • RunLoop的使用场景

    • 实现一个常驻线程；可以开启一个定时器，也可以在子线程中长期监控某些行为；
    • 可以控制定时器在特定模式下执行；

语法知识

static

  • static 修饰局部变量的作用是让局部变量只初始化一次，局部变量在内存中只有一份。

  • static 修饰全局变量，全局变量的作用域仅限于当前文件。

copy

  • copy 返回不可变的字符串对象，mutableCopy返回可变的字符串对象。

  • property里面的copy相当于set方法中使用字符串的copy设置该属性值。
